# jellyfin-tui

The goal of this project is a CMUS-like streaming client for Jellyfin with a featureful TUI.

![image](screen.png)

### Key bindings
|key (alternative)|action|
|---|---|
|Space|play / pause|
|Down (J)|Navigate down|
|Up (K) |Navigate up|
|Right (S)|seek +5s|
|Left (R)|seek -5s|
|N|Next track|
|P|Previous track|
|Tab|Cycle between Artist & Track|
|Shift + Tab|Focus queue|
|Q|Quit|
